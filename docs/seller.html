🏪 คู่มือผู้ขาย

ขายของบน Marketplace

เปิดร้าน จัดการสินค้า/ออเดอร์ ตอบลูกค้าทุกช่องที่เดียว และเชื่อม Facebook

กด Dev quick login → "เข้าเป็นผู้ขาย" ที่หน้าเข้าสู่ระบบ เพื่อเข้าเป็นผู้ขายที่มีร้าน+สินค้าตัวอย่างทันที

1. เป็นผู้ขาย & เปิดร้าน

  1. สมัครเป็นผู้ขาย จากบัญชีผู้ซื้อ กด "เป็นผู้ขาย" — บทบาทอัปเกรดเป็น SELLER (โทเคนจะรีเฟรชให้)
  2. เปิดร้าน ตั้งชื่อร้าน + คำอธิบาย — ผู้ขายหนึ่งคนมีหนึ่งร้าน

2. จัดการสินค้า

  1. เพิ่มสินค้า กรอกชื่อ (ไทย/อังกฤษ) หมวดหมู่ ราคา จำนวนสต็อก และ URL รูป
  2. แก้ไข / ลบ สินค้าได้ตลอด — ราคา/สต็อกอัปเดตเห็นผลทันทีฝั่งผู้ซื้อ

3. จัดการออเดอร์

  1. ดูออเดอร์ร้าน รายการคำสั่งซื้อของร้านคุณ พร้อมกรองตามสถานะ
  2. เลื่อนสถานะ ตามลำดับที่กำหนด: paid_mock → shipped → done (ข้ามขั้นไม่ได้ และแก้ได้เฉพาะออเดอร์ของร้านตัวเอง)

4. กล่องแชตรวม (Omnichannel)

เมนู "แชต" รวมข้อความจากลูกค้าทุกช่องไว้ที่เดียว — มี ป้ายบอกช่อง ว่ามาจากเว็บหรือ Facebook

  1. ลูกค้าเว็บทักจากหน้าสินค้า → เข้ากล่องนี้ (เห็นการ์ดสินค้าที่คุย)
  2. ลูกค้า Facebook ทักมา → เข้ากล่องเดียวกัน มีป้าย FB + ชื่อลูกค้า
  3. คุณตอบจากที่เดียว ข้อความส่งกลับไปยังช่องต้นทางอัตโนมัติ

5. เชื่อม Facebook & sync สินค้า

ในหน้าแดชบอร์ดผู้ขาย มีการ์ด "Facebook (จำลอง)":

  1. เชื่อมเพจ ใส่ Page ID แล้วกดเชื่อม — ผูกเพจกับร้านของคุณ
  2. จำลองข้อความเข้า (ปุ่ม dev) เพื่อทดสอบว่าข้อความ FB เด้งเข้ากล่องแชต
  3. Sync สินค้าไป Facebook กดปุ่มเดียว ดันสินค้าทั้งร้านขึ้น FB catalog + เห็นสถานะจำนวนที่ sync แล้ว
ปัจจุบันการต่อ Facebook เป็นแบบ จำลอง (mock) — สถาปัตยกรรมพร้อมแล้ว จะเสียบ Meta Graph API จริงเมื่อผ่าน Business verification
